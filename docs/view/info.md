# 介绍

View的代码运行在 [Docker容器](https://hub.docker.com/r/rtwadewang/view) 中，该容器基于Linux系统构建，同时参考了View生产环境的配置信息，确保开发和生产环境一致。

## 支持服务

- Apache2
- PHP7.4
- Memcached
- Redis

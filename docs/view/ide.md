# 配合IDE使用

## 关联PHP解释器

### PhpStorm

1.打开偏好设置，选中 `PHP` 配置项，添加新的 `CLI 解释器`

![](/image/view/ide/1.png)

点击 `+` 号添加

![](/image/view/ide/2.png)

2.选中 `From Docker ...` 选项

![](/image/view/ide/3.png)

3.类型选择 `Docker` ，镜像名称选择 `rtwadewang/view`

![](/image/view/ide/4.png)

4.如果PHP信息正确，点击 `确定` 按钮配置成功

![](/image/view/ide/5.png)

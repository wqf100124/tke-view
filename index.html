View本地开发，可以更简单

基于Docker构建的本地开发环境，为你提供更好的开发体验。

开始使用 -> 安装Docker

完整

支持运行Local/Preview/Dev2/RC/Live等环境的代码，方便本地开发和测试。

简洁

无需繁琐的环境配置和安装流程，仅需要几条命令便可轻松搞定。

多态

集成RabbitMQ/Autotest/WSO2/Solr等开发常用的第三方服务。

🏄‍♂️ 运行本地开发环境

docker run -d -p 80:80 -v <本机local代码>:/home/tke/local rtwadewang/view
